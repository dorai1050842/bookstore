회원관리 시스템 구축.

화면구성
- 로그인페이지 : 아이디와 비밀번호 입력받고 로그인 요청이나 회원가입 요청을 함.

minsang9 님 환영합니다.
- 메인페이지 : 로그인한 사용자의 정보를 출력
이 페이지에서 로그아웃요청과 모든회원보기 페이지 요청, 회원수정페이지 요청을
할 수 있음. (모든회원보기, 회원수정, 로그아웃 버튼)

- 회원가입페이지 : 회원정보를 입력하고 가입신청을 할 수 있음.

- 모든회원정보보기페이지 : 모든 회원들의 정보를 열람할 수 있음.
- 회원수정페이지 : 현재 로그인한 회원 정보를 수정할 수 있음.


사용자가 보낼 수 있는 요청들
- main.do : 메인페이지를 달라는 요청
- loginForm.do : 로그인 페이지를 달라는 요청
- login.do : 로그인 처리를 해달라는 요청(파라미터 : id, pw)
- logout.do : 로그아웃 해달라는 요청
- joinForm.do : 회원가입페이지 달라는 요청
- join.do : 회원가입 처리를 해달라는 요청(파라미터 : 회원정보들)
- memberUpdateForm.do : 회원수정 폼을 달라는 요청
- memberUpdatedo : 회원 수정 처리를 해달라는 요청
- memberList.do : 모든 회원보기 페이지를 달라는 요청.
